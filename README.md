A clean and responsive landing page built as part of The Odin Project frontend development curriculum. This project demonstrates modern HTML and CSS techniques used to build a polished homepage layout.

Check it live here https://maryam-20.github.io/odin-landing-page/
## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/odin-landing-page.git
   ```

2. Navigate to the project folder:

   ```bash
   cd odin-landing-page
   ```

3. Open the landing page in your browser:

   - Double-click `index.html`
   - Or serve the folder with a local server and visit `http://localhost:5500`

## Usage

This project is a static landing page template.
- Open `index.html` in a browser to view the page.
- Customize text, images, colors, and layout in the HTML and CSS files.

- HTML5
